# Gymnasium HyperFrames Design

## Style Prompt

Dark technical explainer inspired by Gymnasium's learning-lab feel: precise, calm, and instructional. Use a deep graphite canvas, Gymnasium-like green accents, icy blue state highlights, and amber reward moments. The animation should feel like a teaching console coming alive: clear diagrams, tabular numbers, readable formulas, and measured motion.

## Colors

- Background: `#0d1117`
- Panel: `#151b23`
- Panel border: `#2a3441`
- Primary text: `#f0f6fc`
- Secondary text: `#9fb0c3`
- Gymnasium green accent: `#6cc24a`
- Ice blue state accent: `#58a6ff`
- Reward amber: `#f2b84b`
- Danger hole: `#ef5b5b`

## Typography

- Display and body: `Aptos`, `Segoe UI`, `Microsoft YaHei`, sans-serif
- Code and numeric labels: `Cascadia Mono`, `Consolas`, monospace
- Use strong weight contrast: 800 or 900 for headlines, 300 to 400 for body.

## Motion

- Medium-energy technical motion.
- Use push, scan, and grid-reveal transitions.
- Entrances should combine small y/x movement, opacity, and scale.
- Keep pacing educational: formulas need hold time; data changes should be obvious.

## What NOT to Do

- Do not use purple-blue gradient tech cliches.
- Do not use tiny code or dense unreadable tables.
- Do not make scene jumps without transitions.
- Do not over-decorate with glowing orbs or abstract blobs.
- Do not rely on random motion; every animation should teach the Q-learning concept.
