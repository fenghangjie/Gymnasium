The Update

四行代码，把未来价值传回来

best_next_value = max(Q[next_state])

下一格里最好的动作价值是多少？

old_value = Q[state, action]

旧经验认为，这一步值多少？

learned = reward + gamma * best_next

当前奖励，加上折扣后的未来价值。

new = old + alpha * (learned - old)

让旧 Q 值朝新目标靠近。
